# WebX 2026

## イベント概要

| 項目 | 内容 |
|------|------|
| 正式名称 | WebX 2026（ASIA'S LEADING WEB3 CONFERENCE） |
| 開催日 | 2026年7月13日（月）〜14日（火） |
| 開催地 | 東京（日本） |
| 規模 | アジア最大級のWeb3・暗号資産カンファレンス |
| 公式サイト | https://webx-asia.com/ |

### ステージ構成
- **CRYLステージ** — メインステージ・基調講演
- **Binanceステージ** — ステーブルコイン・決済・市場分析
- **Limitlessステージ** — DeFi・規制・トークン化
- **Visionaryステージ** — スタートアップ・個別戦略発表
- **金融セミナー** — 専門家向け実務セミナー
